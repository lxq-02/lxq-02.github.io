1.1.2、频率与概率

设随机事件 A 在 n 次实验中发生了 m 次，则比值 $\frac{m}{n}$叫做随机事件 A 的相对频率（简称频率），记作$f_{n}(A)$，用公式表示 $\displaystyle f_{n}(A)=\frac{m}{n}$

任何随机事件的频率是介于0和1之间的一个数： $0\leq f_{n}(A) \leq 1$

由随机事件的频率的稳定性可知，随机事件在试验中发生的可能性的大小可以用一个数字来表示。这个刻画随机事件A在试验中发生的可能性大小的、介于0与1之间的数字 p 叫做随机事件A的概率。记作 P(A),即P(A)=p （0<p<1)

1.1.3、事件间的关系

（1）包含

$A\subset B$ A发生必导致B发生

$\phi \subset A \subset \Omega$

包含是集合之间的关系

相等：$A\subset B \ ,\ B\subset A \Rightarrow A=B$

（2）并（和）

$A \cup B=A+B,A与B中至少有一个发生$

（3）交（积）

$A\cap B=AB \ \ \ A,B同时发生$

$AB \cup A,AA=A,A \phi=\phi ,A\Omega=\Omega$

无限可列个：能按某种规律排成一个序列

​ ①自然数： 0，1，2，3，4，······

​ ②整数：0，1，-1，2，-2，·······

​ ③有理数：$\frac{p}{q}$

排不了：①实数（0，1]，②直线点集

（4）差

$A-B, A发生而B不发生， 属于A不属于B$

$A-B=A-AB=A\overline{B}$

（5)互不相容事件

$A,B不同时发生，\ \ \ AB=\phi$

$n个事件A_1,A_2,···A_n,\ A_iA_j=\phi $

（6）对立事件

A,B互不相容，且$A\cup B=\Omega$

$AB=\phi 且 A+B=\Omega$

$A= \bar{B} \ 或者\ B=\bar{A}$

联系与区别：

  • 两事件对立，一定互不相容
  • 互不相容，适用于多个事件
    对立适用于两个事件
  • 互不相容：不能同时发生，可以都不发生
    对立，有且只有一个发生

（7）完备事件组

$A_{1},A_{2},…,A_{n}两两互不相容，且 \mathop{U}^{n}\limits_{i=1}{Ai}=\Omega$

运算规律

  1. 交换律：
    $$
    A\cup B=B\cup A \
    AB=BA
    $$

  2. 结合律：
    $$
    （A \cup B）\cup C=A \cup (B\cup C)\
    (AB)C=A(BC)
    $$

  3. 分配律：
    $$
    A(B\cup C)=AB \cup AC\
    A\cup(BC)=(A\cup B)(A\cup C)
    $$

  4. 德摩根（De Morgn）定律

$$
\overline{A \cup B}=\overline{A}\overline{B}\
\overline{AB}=\bar{A}\cup \bar{B}
$$

1.2.2、古典概率模型

条件：（1）有限个样本

​ （2）等可能性

$\displaystyle P(A)=\frac{A的有利样本点}{\Omega中样本点总数}=\frac{A中包含的基本事件数}{基本事件总数}$

记数

排列组合

加法原理：几种方案（几件上衣） 加法

乘法原理：分几步（上衣和裤子）乘法

排列：

  1. 不重复排列
    从n个不同元素中取出m个不同排列
    $P_{n}^{m}=n(n-1)(n-2)···(n-m+1)$
    $P_{10}^{5}=10 \times 9\times8 \times 7\times 6=\frac{10!}{5!}$
    全排列
    $P_{n}^{n}=n\times (n-1)···3\times 2\times 1=n!$
    $0！=1$
    $0^{0}无意义（a^{0}=1,当a\neq 0时）$
    $5^{0}=\frac{5^{1}}{5^{1}}=1$

  2. 重复排列

    从n个 不同元素中取6m个排列
    $n\times n\times ···\times n=n^{m}$

    组合：从n个不同元素中取出m个不同元素
    $C_{n}^{m}=\frac{P_{n}^{m}}{m!}=\frac{n(n-1)···(n-m+1)}{m(m-1)···2\times 1}=\frac{n!}{m!(n-m)!}$

    $C_{n}^{m}=C_{n}^{n-m} \ \ \ \ \ C_{n}^{0}=C_{n}^{n}=1$

古典性质

（1）非负性 $0\leq P(A)\leq1$

（2）规范性：P（$\Omega$）=1 P（$\phi$）=0

（3）有限可加：A1······An互不相容
$P(A_{1}+A_{2}+···+A_{n})=P(A_1)+···+P(A_n)$

缺点：

  1. 有限个结果
  2. 等可能性

1.2.4、频率与概率

设随机事件 A 在 n 次实验中发生了 m 次，则比值 $\frac{m}{n}$叫做随机事件 A 的相对频率（简称频率），记作$\omega_{n}(A)$，用公式表示 $\omega_{n}(A)=\frac{m}{n}$

性质：

（1）非负性：$0\leq \omega_{n}(A) \leq 1$

（2）规范性：$\omega_{n}(\Omega)=1 \ \ \ \ \omega_{n}(\phi)=0$

（3）可加性 ：$A_{1}···A_{m}不相容\ \omega_{n}(A_{1}+···+A_{m})=\omega_{n}(A_{1})+···+\omega_{n}(A_{m})$

由随机事件的频率的稳定性可知，随机事件在试验中发生的可能性的大小可以用一个数字来表示。这个刻画随机事件A在试验中发生的可能性大小的、介于0与1之间的数字 p 叫做随机事件A的概率。记作 P(A),即P(A)=p （0<p<1)

1.2.5、公理化

描述：古典、几何、统计

①非负②规范③可加

公理：公认的，不需要证明的

  • 公理一（非负性）：$0\leq P_{n}(A) \leq 1$
  • 公理二（规范性）：$P(\Omega)=1$
  • 公理三（完全可加性）：$A_{1}A_{2}······不相容 \P(A_{1}+A_{2}+···)=P(A_{1})+P_{A_2}+···$

证明：

性质一：$P(\phi)=0\ $

$\Omega = \Omega+\phi+\phi+\phi+…….$

$P(\Omega)=P(\Omega+\phi+\phi+\phi+…….)=P(\Omega)+P(\phi)+P(\phi)+…….$

$P(\phi)+P(\phi)+P(\phi)+……=0$

$P(\phi)\geq0 \ \ \ \ P(\phi)=0$

性质二：有限可加性 $A_{1}…A_{n}不相容，P(A_{1}+…+P_n)=P(A_1)+…+P(A_n)$

证明：$A_{1}…A_{n},\phi,\phi,…不相容，\P(A_{1}+…+P_n)=P(A_{1}+…+A_{n}+\phi+\phi+…)=P(A_1)+…+P(A_n)+P(\phi)+P(\phi)+…$

性质三：$P(\bar{A})=1-P(A) $

证明：$A\cap\bar{A}=\phi \ \ \ \ A+\bar{A}=\Omega \ P(\Omega)=P(A+\bar{A})=P(A)+P(\bar{A})=1$

推论：$A_{1}···A_{n}是完备事件组\两两互不相容\并是\Omega，P(A_1)+…+P(A_n)=1$

$P(\Omega)=P(A_1+…+A_n)=P(A_1)+…+P(A_n)=1$

性质四：$①P(A-B)=P(A)-P(AB)\②A\supset B,0\leq P(A-B)=P(A)-P(B) \ \ \ \ 且P(A)\geq P(B)$

证明：$①A=(A-B)\cup AB\ \ \ \ A-B与AB不相容\P(A)=P(A-B)+P(AB) \ \ \ \ P(A-B)=P(A)-P(AB)$

性质五：（加法）$P(A+B)=P(A)+P(B)-P(AB)\A,B互不相容，P(A+B)=P(A)+P(B)$

​ 证明：$A+B=A+(B-AB)\P(A+B)=P(A)+P(B-AB)=P(A)+P(B)-P(AB)\P(B-AB)=P(B)-P(B\cap(AB))=P(B)-P(AB)$

​ $补充：P(A+B+C)=P(A)+P(B)+P(C)-P(AB)-P(AC)-P(BC)+P(ABC)$
