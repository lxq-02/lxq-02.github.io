1.1.11、浏览器访问本地 HTML 页面与远程 HTTP 页面的区别

（1）file 协议是访问你本机的文件资源

​ file协议用于访问本机计算机中的文件，就如同在 Windows 资源管理区中打开文件一样，注意它是针对本地（本机）的。

​ file 协议只能在本地访问。file 只是简单的请求本地文件，与在 Windows 中找到文件，右键，打开方式，用浏览器打开看到的效果是一致的

（2）HTTP 是搭建了服务器

​ HTTP访问本地 HTML，是在本地起了一台 HTTP 服务器，然后你访问自己电脑上的本地服务器，HTTP服务区再去访问你本机的文件资源。

​ 本地搭建了 HTTP 服务器，开放端口之后，别人也可以通过 HTTP 访问到你电脑里的页面，但是 file 协议做不到

1.4.3、HTTP的操作过程：从输入网址到获取页面发生了什么？

（net.cn——>www.net.cn(301)——>https://wanwang.aliyun.com/)

把域名解析位IP

(1)浏览器分析URL，搜索自己的页面缓存

(2)查询hosts文件(本机DNS服务)，是否有对应的特定IP

(3)浏览器搜索自己的DNS缓存，解析目标IP地址

(4)发起一个本地DNS系统调用

(5)浏览器以一个随机端口(软件端口号)向目标server发起了TCP连接请求，建立TCP连接(默认80端口)

(6)如果发起域名不是www开头，Sever给浏览器响应一个301重定向

(7)Web服务器收到请求，分析http header和http body,找出路由及参数，后端程序开始运行

返回一个值200成功404地址不对500服务器的API、错了

(8)后端程序给出一个return (HTML、json、字符串、int)

(9)负载均衡服务器将http请求发给反向代理服务器web集群

把返回的结果存到缓存

(10)Web服务器将结果返回给浏览器，并返回一个http响应(将HTML发送给浏览器)，释放TCP连接

(11)浏览器将结果放入自己的浏览器缓存

(12)浏览器分析结果，构建dom树

渲染

(13)构建渲染树

(14)布局渲染树

(15)绘制到屏幕上(将Web页面显示在显示器的当前浏览器的当前标签页中)

2.2.2、正斜杠（/）与反斜杠（\）

正斜杠（forward slash）（/) 反斜杠（backslash）（\）
在Windows中，正斜杠/表示除法，用来进行整除运算 Windows文件浏览器（资源管理器）上使用 反斜杠 \
在unix系统中，/表示目录 在Windows系统中，\用来表示目录
由于web遵循unix命名，所以在网址（URL）中，/表示目录 \双反斜线 在Windows里表示绝对地址的第一项，比如后面跟上IP地址的话，就是需要打开这个IP地址的网络共享资源
http就是正斜杠 反斜杠”\“是电脑出现了之后为了表示程序设计里的特殊含义才发明的专用标点，就是说，除了程序设计领域外，任何地方你都不应该使用反斜杠的时候，请永远使用正斜杠”/“

2.2.4、Web图片格式

基本的PS技能、jpeg、png、bmp、gif、svg、webgl、h5 canvas

名称 简介 适合场景
（1）JPEG 第一套静态图像压缩标准（11级压缩）、有损压缩 存储照片颜色丰富的复杂图像、广告、轮播、logo
（2）PNG 背景透明、体积小、无损压缩、PNG格式有8位、24位、32位三种格式 头像、图标
（3）Gif 简单动画、颜色少（256种）、无损压缩 小动画
（4）其他旧格式 BMP、TIFF等
（5）WebP 同时提供了有损压缩与无损压缩的图片文件格式、派生自土坯昂编码格式VP8 暂不成熟
（6）SVG 无损缩放矢量图形、SVG图片由直线和曲线以及绘制他们的方法组成 绘制企业Logo、icon等
（7）WebGL 网页3D绘图、JavaScript和OpenGL ES 2.0结合 （TreeJS) 3D机房、浏览器里借助系统显卡，流程显示3D场景和模型
（8）H5 Canvas H5画布，<canvas>有一个基于JavaScript的绘图API 手写签名
